基於圖神經網路於惡意程式偵測研究

Open Source

碩士論文研究主題為惡意程式偵測技術， 利用圖神經網路分析程式之圖結構資料， 以提升惡意程式分類與辨識的準確度。

  • 研究並比較 Graph Convolutional Network (GCN) 與 Graph Attention Network (GAT) 等深度學習模型， 以及傳統機器學習模型在惡意程式分類上的表現。
  • 將程式行為與結構轉換為圖形資料， 並進行資料處理與特徵分析。
  • 研究成果發表於 ICIM 2024 並獲得 最佳論文獎。
  • 將研究程式與資料集開源於 GitHub。
